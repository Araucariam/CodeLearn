Qu'est-ce que Go ?

Go (ou Golang) est un langage de programmation open source créé par Google en 2009. Il a été conçu par Robert Griesemer, Rob Pike et Ken Thompson pour être simple, performant et adapté aux systèmes modernes.

info
Pourquoi Go ?
Go combine la performance du C avec la simplicité de Python. Il est particulièrement adapté pour les microservices, les API REST et les outils CLI.

Installation

Téléchargez Go depuis go.dev/dl, puis vérifiez l'installation :

go version
# go version go1.22.0 linux/amd64

Hello World

Créez un fichier main.go :

go 
package main

import "fmt"

func main() {
    fmt.Println("Bonjour, Go !")
}

Exécutez-le :

go run main.go
Bonjour, Go !

Points forts de Go

  • Compilation rapide — Compilation en quelques secondes
  • Typage statique — Détection des erreurs à la compilation
  • Goroutines — Concurrence légère et efficace
  • Garbage collector — Gestion automatique de la mémoire
  • Un seul binaire — Déploiement simplifié
  • Bibliothèque standard riche — HTTP, JSON, tests intégrés

Variables et types

go 
package main

import "fmt"

func main() {
    // Déclaration explicite
    var nom string = "Alice"
    var age int = 25
    
    // Déclaration courte (inférence de type)
    langage := "Go"
    pi := 3.14159
    estDev := true

    fmt.Printf("%s a %d ans et code en %s\n", nom, age, langage)
    fmt.Println("Pi =", pi, "| Dev:", estDev)
}

Astuce Go
Utilisez := pour les déclarations courtes à l'intérieur des fonctions. C'est la manière idiomatique d'écrire du Go.
