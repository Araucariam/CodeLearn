Qu'est-ce que le HTML ?

HTML (HyperText Markup Language) est le langage de balisage standard pour créer des pages web. Il décrit la structure d'une page web à l'aide de balises (tags).

info
À savoir
HTML n'est pas un langage de programmation. C'est un langage de balisage qui structure le contenu.

Votre première page HTML

Voici la structure minimale d'un document HTML :

html 
<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Ma première page</title>
</head>
<body>
    <h1>Bonjour le monde !</h1>
    <p>Ceci est mon premier paragraphe.</p>
</body>
</html>

Explication ligne par ligne

  • <!DOCTYPE html> — Déclare le type de document (HTML5)
  • <html> — L'élément racine de la page
  • <head> — Contient les métadonnées (titre, encodage, CSS...)
  • <body> — Contient le contenu visible de la page

Les concepts clés

Balises et éléments

Une balise HTML est composée d'un nom entouré de chevrons. La plupart des éléments ont une balise ouvrante et une balise fermante : 

<p>Ceci est un paragraphe</p>
<strong>Texte important</strong>

Attributs

Les attributs fournissent des informations supplémentaires sur un élément : 

<a href="https://example.com" target="_blank">
    Visitez ce site
</a>

<img src="photo.jpg" alt="Description de l'image">
lightbulb
Bonne pratique
Utilisez toujours l'attribut alt sur les images pour l'accessibilité et le référencement.

Outils nécessaires

Pour commencer à écrire du HTML, vous avez besoin de :

  1. Un éditeur de code — VS Code, Sublime Text, ou même le Bloc-notes
  2. Un navigateur web — Chrome, Firefox, Edge
  3. De la curiosité et de la patience 😄
warning
Attention
N'utilisez jamais Microsoft Word pour écrire du HTML ! Les traitements de texte ajoutent du formatage invisible qui casse votre code.

Exercice pratique

Créez un fichier index.html avec le contenu suivant et ouvrez-le dans votre navigateur :

html 
<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Mon Portfolio</title>
</head>
<body>
    <h1>Mon Portfolio</h1>
    <p>Bienvenue sur mon site personnel.</p>
    
    <h2>À propos</h2>
    <p>Je suis développeur web en formation.</p>
    
    <h2>Mes compétences</h2>
    <ul>
        <li>HTML</li>
        <li>CSS</li>
        <li>JavaScript</li>
    </ul>
</body>
</html>

Résumé

Concept Description
<!DOCTYPE> Déclaration du type de document
Élément Balise ouvrante + contenu + balise fermante
Attribut Information supplémentaire sur un élément
<head> Métadonnées du document
<body> Contenu visible de la page
