Qu'est-ce que JavaScript ?

JavaScript est un langage de programmation qui permet de rendre les pages web interactives. C'est le troisième pilier du développement web, avec HTML (structure) et CSS (style).

info
Le saviez-vous ?
JavaScript a été créé en seulement 10 jours par Brendan Eich en 1995. Aujourd'hui, c'est le langage le plus utilisé au monde !

Où s'exécute JavaScript ?

JavaScript peut s'exécuter dans deux environnements :

  • Le navigateur — Chrome, Firefox, Safari (moteur V8, SpiderMonkey...)
  • Le serveur — avec Node.js ou Deno

Premier script

Dans la console du navigateur

Ouvrez la console de votre navigateur (F12 → onglet Console) et tapez :

console.log("Bonjour JavaScript !");
Bonjour JavaScript !

Dans un fichier HTML

html 
<!DOCTYPE html>
<html>
<body>
    <h1 id="titre">Bonjour</h1>
    <button onclick="changerTexte()">Cliquez ici</button>

    <script>
        function changerTexte() {
            const titre = document.getElementById("titre");
            titre.textContent = "JavaScript fonctionne !";
        }
    </script>
</body>
</html>

Les types de données

JavaScript possède plusieurs types de données fondamentaux :

// String (chaîne de caractères)
const nom = "Alice";

// Number (nombre)
const age = 25;
const pi = 3.14159;

// Boolean (booléen)
const estDev = true;

// Array (tableau)
const langages = ["HTML", "CSS", "JS"];

// Object (objet)
const personne = {
    nom: "Alice",
    age: 25,
    langages: ["JS", "Go"]
};

console.log(personne.nom);      // "Alice"
console.log(langages[0]);      // "HTML"
console.log(typeof age);       // "number"

Conseil
Utilisez const par défaut, et let uniquement si la variable doit changer. N'utilisez jamais var dans du code moderne.

Erreur courante
Ne confondez pas = (assignation) avec === (comparaison stricte) ! 
let x = 5;        // Assignation
x === 5;           // Comparaison → true
x === "5";         // Comparaison → false
